## 1. СУТЬ @LEADYCRM_bot

Один бот — два интерфейса:

| Аудитория | Зачем | Что делает |
|-----------|-------|------------|
| **Owner (клиент)** | Витрина + поддержка | Покупает @CompanyBot, пишет тикеты |
| **Platform Manager** | Саппорт | Отвечает на тикеты, смотрит компании, статистику |
| **Admin (ты)** | Управление | ВСЁ |

## 2. РОЛИ И МЕНЮ

### 2.1. Owner — главное меню (6 кнопок)

| Кнопка | Действие |
|--------|----------|
| 💳 Подписка | Статус + кнопка Оплатить/Продлить |
| 💬 Поддержка | Тикеты компании + кнопка Создать |
| ⚙️ Настройки | Название, тариф, ID, статусы сделок (Pro) |
| 📊 Статистика | Только Pro. Base → заглушка |

| ❓ Помощь | Список команд |

**Истекшая подписка**: заглушка + кнопка [🔄 Продлить]

| 🏢 Компании | Список, просмотр |
| 📊 Статистика | По компаниям |
| ❓ Помощь | Команды |

### 2.3. Admin — меню (7 кнопок)

| Кнопка | Действие |
|--------|----------|
| 🏢 Компании | Список, карточка, блокировка |

## 3. ОНБОРДИНГ OWNER (6 шагов)

```
/start
  │
  ├─ FSM активен → сброс, начать заново
  │
  ├─ ШАГ 1: CONSENT
  │   ├─ [✅ Принимаю] → Шаг 2
  │   └─ [❌ Отказываюсь] → заглушка, /start → предупреждение + оферта
  │
  ├─ ШАГ 2: ИНФО О LEady (3 сообщения с фото)
  │   1. «Leady — CRM в Telegram» + скрин
  │   2. Возможности: сделки, клиенты, задачи
  │   3. «Создайте компанию за 1 минуту»
  │
  ├─ ШАГ 3: НАЗВАНИЕ КОМПАНИИ
  │   └─ FSM: waiting_company_name (2-100 символов)
  │
  ├─ ШАГ 4: ТАРИФ
  │   ├─ [Базовый — 999 ₽/мес]
  │   └─ [Pro — 2 999 ₽/мес]
  │
  ├─ ШАГ 5: ОПЛАТА (обязательно)
  │   ├─ Ссылка на YooKassa
  │   ├─ Кнопки: [✅ Я оплатил] [🔄 Проверить снова]
  │   └─ Успех → Шаг 6
  │
  └─ ШАГ 6: СОЗДАНИЕ @CompanyBot
      ├─ 6a: Ввод имени бота (RomashkaBot)
      │   └─ Валидация: латиница, заканчивается на bot, 5-32 символа
      ├─ 6b: Инструкция для @BotFather
      │   └─ Бот показывает точные команды для копирования
      └─ 6c: Ввод токена → проверка через getMe → активация
```

### Ключевые правила /start:

- **FSM активен** → сброс, начать с consent
- **Отказ от consent** → заглушка. /start → предупреждение + оферта
- **Уже есть компания** → главное меню с приветствием «Добро пожаловать, Имя!»
- **Оплата не прошла** → кнопка повтора, FSM на шаге оплаты
- **Токен невалиден** → сообщение об ошибке, можно ввести заново

## 7. PLATFORM MANAGER FLOW

```
Admin → /admin → «Пригласить менеджера»
  └─ deep-link: t.me/LEADYCRM_bot?start=invite_UUID

Platform Manager → переход по ссылке
  └─ Статус: pending_approval
  └─ Видит: «Ожидайте подтверждения»

Admin → /admin → «На авторизации»
  ├─ Список pending
  ├─ [✅ Допустить] → active → меню
  └─ [❌ Отказать] → заблокирован
```
